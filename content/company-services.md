# renameDP 전기시공버전

## 서비스 요약
전기시공 현장의 데이터 기록·보고를 디지털화하는 플랫폼입니다.
전봇대 등 구조물 촬영 시 전산화 번호를 AI로 자동 인식하고, 사진·시공 정보를 체계적으로 관리하여
수기 작업을 없애고 현장 업무 효율을 높입니다.

## 주요 기능
- **전산화 번호 자동 추출**: 촬영한 구조물 사진에서 번호찰의 전산화 번호를 OCR로 자동 인식, 수기 입력 없이 빠르고 정확한 데이터 처리
- **현장 사진 업로드**: 촬영한 모든 사진을 한 번에 서버에 안전하게 업로드, 언제 어디서나 간편하게 등록 가능
- **AI 메신저**: 현장 채팅 내용을 매일 자동 정리하여 일일보고로 제공, 개인 메신저 의존 없이 현장 소통 체계화
- **프로젝트 대시보드**: 프로젝트 진척률·일정·작업 위치·계약 금액·진행 분포를 한눈에 확인하여 빠른 의사결정 지원
- **시공정보 관리**: 작업별 세부 내용과 전산화 번호 등 적재 데이터를 보완·수정하고 구조물별 사진 정보를 체계적으로 관리
- **프로젝트 관리**: 다양한 시공 범위를 아우르는 프로젝트 등록과 관리를 한곳에서 처리
- **Zip 파일 일괄 업로드**: OCR로 전산화 번호 자동 입력 및 관련 구조물끼리 그룹화하여 일괄 업로드

## 연관 키워드
전기, 전기공사, 전기시공, 전봇대, 전산화번호, OCR, 번호찰, 시공사진, 현장사진, AI메신저, 일일보고, 작업일보, 디지털전환, 현장관리, 공정관리

# renameDP 종합건설버전

## 서비스 요약
건설현장의 분절된 프로세스를 통합 데이터 기반 관리로 전환하는 현장 디지털화 플랫폼입니다.
수기 작업을 AI로 자동화하여 현장-관리 간 실시간 소통을 지원하고, 시공 품질 투명성 확보 및
안전 리스크 선제 대응을 통해 건설현장의 행정 효율과 안전 수준을 동시에 높입니다.

## 주요 기능
- **시공 품질 증빙 시스템**: 촬영 시 장소·시간 등 메타데이터가 자동 매핑되어 업로드되며, 카테고리에 맞게 파일명 자동 생성
- **실시간 현장 소통 및 관리**: 현장별 채팅으로 실시간 업무 보고, AI 일별 요약으로 시공 현황 조회, 도면·메모 첨부 및 TBM 사항 전파·서명
- **위험성평가 및 대응**: 현장별 맞춤 위험성평가·TBM 전자서명 제출, 위험요소 신고·대응 프로세스 단축, 중대재해 법적 증거 마련
- **현장별 프로젝트 관리**: 공종·공정·작업 기한 관리, 현장별 인력·일정·데이터 관리, 문서·공정표·도면·위치 태그 통합 관리
- **AI 작업 일보 자동 생성**: 업로드된 사진·문서 기반 AI 작업일보 자동 작성, 고객 요구사항에 맞는 양식 커스터마이징
- **CCTV 실시간 위험 감지 (옵션)**: AI CCTV 실시간 분석으로 이상 상황 즉시 탐지, 안전 이벤트 자동 인식·관리, 과거 영상 기록 조회
- **ERP·내부 관리 기능**: 프로젝트 대시보드(기성률·실행률·자금 현황), 전자결재, 법인별 권한 분리, 근태 관리, 전사 게시글, 운영자 설정 페이지
